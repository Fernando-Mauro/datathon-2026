Visual rules
  • Pitch-black canvas, no gradients except HAVI
  • Spanish (Mexican) tú-form copy, no emoji
  • Wordmark always lowercase with comma
  • 40–48px icon chips, 16% tint, stroke glyph
  • Pill buttons for primary actions
  • Flat — no shadows, depth via tint shifts
  • HAVI ring: only conic gradient in the system
